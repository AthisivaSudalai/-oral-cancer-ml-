# Oral Cancer Prediction — ML Pipeline Project

## Overview
Comparative analysis of ML algorithms for oral cancer prediction using tabular clinical data with SHAP-based explainability.

## Dataset
Ankush Pandey — Oral Cancer Prediction Dataset (Kaggle, MIT License)
25 features, ~85k rows, synthetic clinical data compiled from WHO and NCI statistics.

## Models Compared
- Logistic Regression
- Random Forest
- XGBoost
- LightGBM

## Project Structure
- data/ — raw dataset
- notebooks/ — one notebook per phase
- models/ — saved trained models
- outputs/ — preprocessed data and result tables
- reports/ — final report

## Purpose
Pipeline practice project to build tabular clinical ML skills before working with the SWAN dataset for Menopausal Syndrome research.
